Стек:
React 18
TypeScript
Vite

UI и стилизация:
Mantine UI (v7) - полная библиотека компонентов
@mantine/hooks - кастомные хуки

Маршрутизация:
React Router DOM v6 - клиентская маршрутизация
BrowserRouter - обертка для SPA
Routes, Route - декларативная маршрутизация
Navigate - программное перенаправление

Карты и визуализация:
React-Leaflet / Leaflet
Google Maps API
D3.js или Recharts (для графиков аналитики)

Состояние:
React Context API / Zustand / Redux Toolkit
Локальное состояние через useState, useReducer

HTTP клиент:
Axios или Fetch API (для работы с бэкендом)
React Query / SWR

Валидация форм:
React Hook Form + Zod / Yup

Аутентификация:
JWT токены

Локальное хранение в localStorage/sessionStorage
