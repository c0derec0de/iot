## Технологический стек

### Основные технологии
- **React 18** - библиотека для построения пользовательских интерфейсов
- **TypeScript** - язык программирования для статической типизации
- **Vite** - инструмент сборки и development server

### UI и стилизация
- **Mantine UI (v7)** - полная библиотека компонентов
  - `@mantine/core` - основные UI компоненты
  - `@mantine/hooks` - кастомные React хуки

### Маршрутизация
- **React Router DOM v6** - клиентская маршрутизация
  - `BrowserRouter` - обертка для одностраничного приложения
  - `Routes`, `Route` - декларативная система маршрутов
  - `Navigate` - программное перенаправление
  - `ProtectedRoute` - кастомный компонент защиты маршрутов

### Карты и визуализация данных
- **React-Leaflet / Leaflet** - библиотека для интерактивных карт
- **Google Maps API** - альтернативный сервис карт
- **Recharts** - библиотека для построения графиков и диаграмм аналитики

### Управление состоянием
- **React Context API** - встроенная система контекста React
- **Zustand / Redux Toolkit** - внешние библиотеки управления состоянием
- Локальное состояние через `useState` и `useReducer`

### Работа с HTTP
- **Axios** или **Fetch API** - HTTP клиенты для взаимодействия с бэкендом
- **React Query / SWR** - библиотеки для кэширования и управления запросами

### Валидация форм
- **React Hook Form** - библиотека для работы с формами
- **Zod / Yup** - схемы валидации данных

### Аутентификация и безопасность
- **JWT токены** (JSON Web Tokens) - механизм аутентификации
- Локальное хранение в `localStorage`/`sessionStorage` - хранение токенов и данных сессии

### Инструменты разработки
- **ESLint** - линтинг JavaScript/TypeScript
- **Prettier** - форматирование кода
- **Git** - контроль версий
